# KnowMe iOS

![221327599-911f17b1-272a-4cf2-afe5-116dab1ec68f](https://github.com/abhirajbhattashali/KnowMe-iOS-/assets/60516848/1b43594d-84b4-4bba-b867-760781e4dfd4)

App developed for Next Tech Lab , SRMAP University, specifically for iPhone users to detect objects located in the lab using Image Recognition. The app has been designed to be intuitive and user-friendly, and it will be available to all students and members at the Lab.

<p> The IMAGE RECOGNITION model is trained using Create ML framework and integrated into the app using CoreML framework.</p>
<p>The app features Realtime Scanner to scan objects in realtime as well as Static Scanner allowing users to capture images from their camera, upload it in the app and recognize the lab object present in the image , in case of no match , blank is displayed on user screen.</p>

<p>Technologies used - Core ML , SwiftUI , Swift5 , Create ML, XCode</p>
